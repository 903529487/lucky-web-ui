# lucky-web-ui

本组件库依赖 lucky-web、react @17.0.0 以上

安装依赖

```bash
$ npm i
```

启动文档

```bash
$ npm start
```

打包文档

```bash
$ npm run docs:build
```

打包组件库

```bash
$ npm run build
```

### 按需加载

```tsx | pure
import Img from 'lucky-web-ui/es/Components/Img';

<Img />;
```

### 全部引入

```tsx | pure
import { Img } from 'lucky-web-ui';
<Img />;
```
